# skills

Personal collection of [agent skills](https://github.com/vercel-labs/skills) for Claude Code and friends.

Install:

```sh
pnpm dlx skills add jergason/skills
```

## Skills

| Skill | What it does |
| --- | --- |
| `data-layer-reviewer` | Review data layer code for pattern compliance, security, and best practices. |
| `diagramming-flows` | Render mermaid diagrams as ASCII/unicode art directly in the terminal. |
| `profiling-user-flows` | Profile user flows and write up performance reports. |
| `readwise-digest` | Pull saved articles via the Readwise CLI, group by theme, and append a digest to the weekly lab notebook. |
| `release-notes` | Generate release notes from PRs and GitHub assets. |
| `seppuku` | For when the agent has brought dishonor upon itself and must exit the stage gracefully. |
| `stacked-prs` | Create and manage stacked pull requests. |
| `twitter-digest` | Filter and digest Twitter bookmarks. |
